## はじめに

まず、開発サーバーを実行します。

```bash
npm run dev
# または
yarn dev
```

ブラウザで[http://localhost:3000](http://localhost:3000)を開いて、結果を確認してください。

`pages / index.js`を変更することで、ページの編集を開始できます。 ファイルを編集すると、ページが自動更新されます。
